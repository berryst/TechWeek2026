# Activity 16: 🍳 Recipe Remix Lab

[← Back to Activities](../README.md)

| | |
|---|---|
| **Time** | 5 min |
| **Audience** | Years 3–8 |
| **Skill** | Constraint prompting + cultural fusion |
| **Tool** | Copilot (text + image) |

> **Why it works:** Food is a universal language. This activity brings whānau into the room without anyone leaving home — and shows AI working with constraints.

## Step-by-step lab

1. Choose 3 ingredients that you really have at home.
2. Use Prompt 1 to ask Copilot for a simple recipe using those ingredients.
3. Use Prompt 2 to remix the recipe with a cultural twist.
4. Use Prompt 3 to generate an image of the finished dish.
5. Decide whether you would like to make it at home with an adult.
## Prompt template

```text
PROMPT 1 (recipe):Invent a simple recipe a 10-year-old could make with help from a parent,using these 3 ingredients: [ING 1], [ING 2], [ING 3].Give it a fun name. Use simple steps. Cooking time under 30 minutes.PROMPT 2 (cultural twist):Now remix this recipe with a [Samoan / Tongan / Cook Islands / Fijian / Niuean] twist.Add one traditional ingredient and explain what it adds to the flavour.PROMPT 3 (image):A warm cartoon illustration of the finished dish on a wooden plate,served on a tropical leaf, top-down view, bright colours, mouth-watering.
```
