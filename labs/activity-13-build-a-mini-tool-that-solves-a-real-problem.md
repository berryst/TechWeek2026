# Activity 13: 🛠️ Build a Mini Tool That Solves a Real Problem

[← Back to Activities](../README.md)

| | |
|---|---|
| **Time** | 10 min |
| **Audience** | Years 7–8 |
| **Skill** | Problem identification + spec writing + code generation |
| **Tool** | Copilot (text) + Notepad/TextEdit + web browser |

> **Why it works:** This is real product thinking. Students learn to spot a problem in their life and turn it into a working tool — the exact mindset of a Microsoft Program Manager.

## Step-by-step lab

1. Think of one real problem in your week that you want to solve.
2. Fill in the prompt template to describe your problem, the input, and the output you want.
3. Paste the prompt into Copilot and generate your tool.
4. Copy the code, save it as tool.html, and open it in your browser.
5. Test whether your tool really helps with your problem.
6. Ask Copilot to add one more feature and improve your tool.
## Prompt template

```text
I want to build a simple tool that solves this problem in my life:[DESCRIBE THE PROBLEM IN ONE SENTENCE].The tool should:- Let me [INPUT — e.g. type how much I spent today]- Save it (in browser local storage so it remembers next time)- Show me [OUTPUT — e.g. the total this week]- Have a 'Reset' buttonBuild this as a single HTML file with inline CSS and JavaScript.Style it so it looks clean and modern — big buttons, easy to tap on a phone.Use a friendly colour scheme.Give me the full file in one block.
```

**Sample prompt 1**

```text
I want to build a simple tool that solves this problem in my life: I forget my homework tasks. The tool should let me type homework tasks, save them, show me my list, and have a Reset button. Build this as a single HTML file with inline CSS and JavaScript. Style it so it looks clean and modern with big buttons.
```

**Sample prompt 2**

```text
I want to build a simple tool that solves this problem in my life: I want to track how much pocket money I spend. The tool should:- let me type how much I spent today, - save it, - show me the total this week, and - have a Reset button. Build this as a single HTML file with inline CSS and JavaScript. Style it so it looks clean and modern with big buttons.
```
