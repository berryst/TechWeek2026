# Activity 12: 🎮 Make a Mini Quiz Game

[← Back to Activities](../README.md)

| | |
|---|---|
| **Time** | 10 min |
| **Audience** | Years 7–8 |
| **Skill** | App logic + iterative debugging + JavaScript exposure |
| **Tool** | Copilot (text) + Notepad/TextEdit + web browser |

> **Why it works:** Students see code respond to clicks for the first time. The 'aha' moment when a button works is unforgettable — and it teaches them that an app is just data + logic + screen.

## Step-by-step lab

1. Open Notepad or TextEdit and open Copilot in a browser.
2. Pick a quiz topic and write 5 questions with the correct answers.
3. Paste the prompt template into Copilot and add your topic, questions, and answers.
4. Copy the full code, save it as quiz.html, and open it in your browser.
5. Test your quiz and see if it works properly.
6. If something breaks, describe the problem to Copilot, update the code, and test again.
7. Swap with a partner and play each other's quiz.
## Prompt template

```text
Build a simple browser video game as one HTML file with inline CSS and JavaScript.Game idea: [CHOOSE ONE — block stacker, maze chaser, dodger, catch game, endless runner].Theme: [STUDENT'S THEME — e.g. space, rugby, oceans, robots].The game should include:- A start screen with the game title- Easy keyboard controls- A score counter- A game over screen and restart button- Bright colours and simple shapes or spritesKeep it fun, simple, and playable in a browser for students aged 10–13.Give me one full HTML file I can save and open in a browser.
```

**Sample prompt 1 – Bubble Blaster**

```text
Build a simple browser game as one HTML file with inline CSS and JavaScript. Make it a bubble shooting game called Bubble Blaster. The player can aim left or right using the keyboard or by moving the mouse, then shoot bubbles toward the top of the screen by clicking or pressing the space bar. Each bubble popped should add points to the score. Include a start screen, score counter, game over screen, and restart button. Use bright colours, simple bubble shapes, and make it easy for students to play in a browser.
```

**Sample prompt 2 – Space Escape**

```text
Build a simple browser game as one HTML file with inline CSS and JavaScript. Make it a dodger game called Space Escape. The player moves a spaceship left and right with the arrow keys to avoid falling asteroids. Include a start screen, score counter, game over screen, and restart button. Use bright space colours, simple shapes, and make it easy for students to play.
```

**Sample prompt 3 – Kai Quest**

```text
Build a simple browser game as one HTML file with inline CSS and JavaScript. Make it a maze chaser game called Kai Quest. The player moves through a maze collecting fruit while avoiding one enemy. Include easy keyboard controls, a score counter, a win or game over screen, and a restart button. Use colourful Pacific-inspired colours and keep the code simple enough for students to explore. Add the animated sprites, sound effects and multiple levels.
```
